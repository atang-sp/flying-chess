🎲 飞行棋配置导出功能

📋 功能说明

配置导出功能允许用户将游戏配置保存为JSON文件，方便分享和备份。支持以下配置项：

  • 玩家设置 - 玩家数量和姓名配置
  • 惩罚设置 - 工具、身体部位、姿势等惩罚配置
  • 棋盘设置 - 各种格子数量的配置
  • 机关设置 - 机关格子的配置
  • 棋盘布局 - 当前棋盘的完整布局（包含随机种子）

🚀 使用方法

  1. 在游戏中点击左下角的 📤 导出 按钮
  2. 选择要导出的配置项
  3. 点击 导出 JSON 文件 按钮
  4. 浏览器会自动下载配置文件

📄 导出文件格式

导出的JSON文件包含以下结构：

{ "version": "1.0.0", "exportedAt": "2025-07-20T10:30:00.000Z", "gameTitle": "飞行棋配置", "description": "游戏配置导出文件", "data": { "playerSettings": { "playerCount": 2, "playerNames": ["玩家1", "玩家2"] }, "punishmentConfig": { "tools": [...], "bodyParts": [...], "positions": [...], "minStrikes": 10, "maxStrikes": 30, "step": 5, "maxTakeoffFailures": 5 }, "boardConfig": { "punishmentCells": 28, "bonusCells": 1, "reverseCells": 2, "restCells": 1, "restartCells": 4, "trapCells": 2, "totalCells": 40 }, "trapConfig": [...], "boardContent": { "seed": "random_seed_string", "board": [...], "generatedAt": 1721472600000 } } }

🔧 技术特性

  • 支持选择性导出，用户可以选择需要的配置项
  • 自动生成带时间戳的文件名
  • 实时显示导出文件大小和配置项数量
  • 包含数据验证，确保导出内容的完整性
  • 支持棋盘布局快照，包含随机种子
  • 为后续二维码功能预留接口
