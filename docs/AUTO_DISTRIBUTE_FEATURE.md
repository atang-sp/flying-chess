# 棋盘格子自动分配功能

## 功能概述

在棋盘规格配置页面新增了自动分配功能，通过独立的模态框界面，可以根据棋盘总数自动按比例分配各类格子，大大简化了棋盘配置过程。

## 架构设计

### 组件结构
- **主组件**: `EditBoardModal.vue` - 棋盘编辑主界面
- **子组件**: `AutoDistributeModal.vue` - 独立的自动分配模态框
- **测试页面**: `test-modal.vue` - 功能测试页面

## 功能特性

### 1. 预设分配模式

提供三种预设的分配模式：

#### 平衡模式 (推荐新手)
- 惩罚格子：60%
- 空白格子：5%
- 商店格子：8%
- 金币格子：8%
- 奖励格子：5%
- 后退格子：5%
- 前进格子：4%
- 特殊事件格子：5%

#### 惩罚模式 (刺激性更强)
- 惩罚格子：75%
- 后退格子：8%
- 特殊事件格子：5.5%
- 空白格子：5%
- 商店格子：3%
- 金币格子：3%
- 奖励格子：2.5%
- 前进格子：2%

#### 冒险模式 (充满变数)
- 惩罚格子：48%
- 商店格子：12%
- 金币格子：12%
- 特殊事件格子：10%
- 奖励格子：8%
- 前进格子：6%
- 后退格子：4%
- 空白格子：0%



### 2. 智能分配算法

- 按比例计算各类格子数量
- 自动处理向下取整产生的剩余格子
- 剩余格子优先分配给惩罚格子
- 确保分配结果不超过总格子数

## 使用方法

### 基本使用

1. 在棋盘规格配置页面设置格子总数
2. 点击"自动分配"按钮打开分配模态框
3. 在模态框中选择合适的分配模式
4. 查看预览分配结果
5. 点击"确定应用"完成自动分配
6. 可在主界面继续手动微调具体数量



### 快速调整

- 使用"重置"按钮清空所有分配
- 可以在自动分配后手动调整各类格子的具体数量

## 技术实现

### 核心算法

```typescript
// 按比例计算每种格子的数量
for (const [cellType, ratio] of Object.entries(ratios)) {
  const count = Math.max(0, Math.floor(total * ratio))
  allocations[cellType] = count
  allocated += count
}

// 处理剩余格子
const remaining = total - allocated
if (remaining > 0) {
  allocations.punishmentCells += remaining
}
```

### 比例验证

```typescript
// 验证自定义比例总和
const totalPercentage = Object.values(customRatios.value).reduce((sum, val) => sum + val, 0)
if (Math.abs(totalPercentage - 100) > 0.1) {
  // 显示错误提示
}
```

## 用户体验优化

1. **实时反馈**：显示剩余可分配格子数量
2. **状态提示**：用不同颜色提示分配状态（超出/未完成/完成）
3. **智能提示**：在自定义模式下实时显示比例总和状态
4. **一键操作**：提供重置和自动分配的快捷按钮
5. **预设模板**：提供多种预设模式满足不同需求

## 扩展性

该功能设计具有良好的扩展性：

- 可以轻松添加新的预设模式
- 支持动态调整比例算法
- 可以集成更复杂的分配策略
- 支持保存和加载自定义比例配置

## 兼容性

- 完全兼容现有的手动配置方式
- 不影响已有的棋盘配置数据
- 支持在自动分配后进行手动微调
